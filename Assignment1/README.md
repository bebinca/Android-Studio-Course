# Assignment1

## View used

`ImageView`：展示图片

`Button`：切换文本和图片

`TextView`：展示文本

`Switch`：隐藏所有组件/显示所有组件

`ToggleButton`：去往第一张图片/去往最后一张图片

## Environment

开发环境：Android Studio
